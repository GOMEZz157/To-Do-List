# 📝 To-Do List

Aplicação web de **lista de tarefas (To-Do List)** desenvolvida com **HTML, CSS e JavaScript**, permitindo adicionar, editar, pesquisar e gerenciar tarefas diretamente no navegador.

O projeto foi criado com o objetivo de praticar **manipulação do DOM, eventos em JavaScript e armazenamento de dados no navegador utilizando LocalStorage**.

---

# 🚀 Demonstração

Acesse o repositório do projeto:

🔗 https://github.com/GOMEZz157/To-Do-List


---

# ⚙️ Funcionalidades

- ➕ Adicionar novas tarefas
- ✏️ Editar tarefas
- ✅ Marcar tarefas como concluídas
- 🗑️ Remover tarefas
- 🔎 Pesquisar tarefas
- 🎯 Filtrar tarefas (todas, concluídas ou pendentes)
- 💾 Armazenamento das tarefas no **LocalStorage**

---

# 🛠️ Tecnologias Utilizadas

- **HTML5** – Estrutura da aplicação
- **CSS3** – Estilização da interface
- **JavaScript (Vanilla JS)** – Lógica da aplicação
- **LocalStorage** – Persistência de dados no navegador

# 💻 Como Executar o Projeto

Clone o repositório:

```bash
git clone https://github.com/GOMEZz157/To-Do-List.git
```

Entre na pasta do projeto:

```bash
cd To-Do-List
```

Abra o arquivo `index.html` no navegador.
